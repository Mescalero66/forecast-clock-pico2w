### GPS Weather Clock Project
(Australia Only)

## Pins
| Component | Interface | I/O | Pin |
| --- | --- | --- | --- |
| **GPS** | `UART1` | UART0 RX | GP1 |
| **Multiplexer** | `I2C` | I2C1 SDA<br>I2C1 SCL | GP14<br>GP15 |
